# Spanish-Dutch Translation Machine Project

## Documentation

### Features
- Accurate translations from Spanish to Dutch.
- User-friendly interface for easy usage.
- Support for a wide range of vocabulary and phrases.

### How to Use
1. Input a Spanish word or phrase in the designated field.
2. Click the "Translate" button to receive the Dutch translation.
3. Review the translation and make any necessary adjustments.

### Installation Instructions
To install the Spanish-Dutch Translation Machine:
1. Clone the repository:
   ```bash
   git clone https://github.com/Martico14/Corlaer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Corlaer
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

### Project Structure
- `src/`: Contains the source code for the application.
- `public/`: Contains static assets such as images and the index.html file.
- `package.json`: Lists the dependencies and scripts for the project.
- `README.md`: Documentation about the project.
